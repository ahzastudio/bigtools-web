Download Digital Elevation Model Nasional dengan resolusi 0.27 arc-second

Tentang DEMNAS

DEMNAS (Digital Elevation Model Nasional) adalah data elevasi permukaan bumi Indonesia dengan resolusi spasial 0.27 arc-second (~8.3 meter). Data ini sangat berguna untuk analisis topografi, hidrologi, perencanaan wilayah, dan berbagai aplikasi geospasial lainnya.

💡 Info: DEMNAS mencakup seluruh wilayah Indonesia dan merupakan data DEM resolusi tinggi yang disediakan gratis oleh BIG.

Langkah-Langkah Penggunaan

Langkah 1: Buka Tool DEMNAS Downloader

Buka plugin BIG Downloader di QGIS, ArcGIS Pro, atau ArcMap. Pilih tool DEMNAS Downloader dari toolbox.

Langkah 2: Load Grid Index

Tool memerlukan file Grid Index DEMNAS untuk menentukan tile mana yang perlu didownload. Anda bisa:

  • Download Grid Index dari website (tersedia di halaman download)
  • Load file shapefile Grid Index yang sudah Anda download

Langkah 3: Pilih Area Download

Tentukan area yang ingin Anda download DEMNAS-nya:

  • Canvas Extent: Menggunakan area tampilan peta saat ini
  • Layer Extent: Menggunakan extent dari layer yang dipilih
  • Selected Features: Hanya tile yang dipilih di Grid Index
  • Draw on Canvas: Menggambar area secara manual

Langkah 4: Masukkan Token API

Masukkan token API yang sudah Anda dapatkan dari Ina-Geoportal. Token ini diperlukan untuk autentikasi ke server BIG. Lihat Panduan Token API jika belum punya.

Langkah 5: Pilih Opsi Processing (Opsional)

Tool menyediakan beberapa opsi processing:

  • Merge Tiles: Menggabungkan semua tile menjadi satu file raster
  • Clip to AOI: Memotong hasil sesuai area of interest
  • Fill NoData: Mengisi nilai kosong dengan interpolasi

Langkah 6: Tentukan Folder Output

Pilih folder untuk menyimpan file DEMNAS. Tool akan membuat struktur folder otomatis untuk mengorganisir tile-tile yang didownload.

Langkah 7: Jalankan Tool

Klik tombol Run atau OK. Tool akan:

  • Mengidentifikasi tile DEMNAS yang diperlukan
  • Mendownload setiap tile dari server BIG
  • Melakukan processing jika opsi dipilih (merge, clip, dll)
  • Menampilkan progress dan status real-time
  • Menambahkan hasil ke peta

✅ Tips: Untuk area yang luas, gunakan opsi "Merge Tiles" agar mendapatkan satu file seamless yang lebih mudah digunakan untuk analisis.
⚠ Perhatian: Setiap tile DEMNAS berukuran ~50-100 MB. Pastikan Anda memiliki ruang penyimpanan yang cukup dan koneksi internet yang stabil.
💾 Output: File DEMNAS tersimpan dalam format GeoTIFF (.tif) dengan nilai elevasi dalam meter, sistem koordinat WGS84.
🔧 Troubleshooting: Jika download gagal, cek koneksi internet dan pastikan token masih valid. Beberapa tile mungkin tidak tersedia untuk area tertentu.
