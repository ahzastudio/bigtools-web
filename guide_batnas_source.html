Panduan BATNAS Downloader

Download Batimetri Nasional untuk analisis kedalaman perairan Indonesia

Tentang BATNAS

BATNAS (Batimetri Nasional) adalah data kedalaman perairan Indonesia yang mencakup laut, danau, dan sungai besar. Data ini sangat penting untuk navigasi maritim, perencanaan pelabuhan, studi oseanografi, dan analisis potensi sumber daya laut.

💡 Info: BATNAS menyediakan informasi kedalaman perairan dengan resolusi yang bervariasi tergantung area dan metode pengukuran.

Langkah-Langkah Penggunaan

Langkah 1: Buka Tool BATNAS Downloader

Buka plugin BIG Downloader di QGIS, ArcGIS Pro, atau ArcMap. Pilih tool BATNAS Downloader dari toolbox.

Langkah 2: Load Grid Index BATNAS

Sama seperti DEMNAS, tool ini memerlukan Grid Index BATNAS:

  • Download Grid Index BATNAS dari halaman download website
  • Load file shapefile Grid Index ke dalam tool
  • Grid akan menampilkan tile-tile BATNAS yang tersedia

Langkah 3: Pilih Area Perairan

Tentukan area perairan yang ingin Anda download data batimetrinya:

  • Canvas Extent: Area tampilan peta saat ini
  • Layer Extent: Extent dari layer perairan yang dipilih
  • Selected Features: Tile yang dipilih manual dari Grid Index
  • Draw on Canvas: Gambar polygon area secara manual

Langkah 4: Masukkan Token API

Masukkan token API dari Ina-Geoportal. Token yang sama dengan DEMNAS/RBI bisa digunakan. Lihat Panduan Token API.

Langkah 5: Konfigurasi Output

Atur opsi output sesuai kebutuhan:

  • Merge Tiles: Gabungkan tile menjadi satu raster seamless
  • Clip to Coastline: Potong data hanya untuk area perairan
  • Invert Values: Ubah nilai negatif (kedalaman) menjadi positif

Langkah 6: Tentukan Folder Output

Pilih lokasi penyimpanan file BATNAS. Tool akan membuat subfolder untuk mengorganisir tile-tile yang didownload.

Langkah 7: Jalankan Tool

Klik Run atau OK untuk memulai. Proses yang terjadi:

  • Identifikasi tile BATNAS yang overlap dengan area
  • Download tile dari server BIG secara otomatis
  • Processing data (merge, clip, invert) jika dipilih
  • Tampilkan progress bar dan log detail
  • Load hasil ke peta dengan styling kedalaman

✅ Tips: BATNAS sangat berguna untuk analisis rute pelayaran, studi tsunami, dan perencanaan infrastruktur pesisir. Kombinasikan dengan DEMNAS untuk analisis coastal zone yang komprehensif.
⚠ Perhatian: Tidak semua area perairan Indonesia memiliki data BATNAS. Coverage terbatas pada area yang sudah dilakukan survei batimetri.
💾 Output: File BATNAS dalam format GeoTIFF dengan nilai kedalaman (biasanya negatif, di bawah permukaan laut). Gunakan opsi "Invert Values" untuk mendapatkan nilai positif.
🔧 Troubleshooting: Jika tile tidak tersedia, cek Grid Index untuk melihat coverage area. Beberapa perairan dalam (deep ocean) mungkin belum memiliki data detail.
📊 Visualisasi: Gunakan color ramp "Blues" atau "Ocean" untuk visualisasi yang lebih intuitif. Nilai lebih gelap = lebih dalam.
