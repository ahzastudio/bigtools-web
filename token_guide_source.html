Cara Mendapatkan Token API

Token diperlukan untuk mengakses data DEMNAS, BATNAS, dan RBI dari server Tanah Air.

Langkah 1: Buka Halaman Login

Buka halaman Login Ina-Geoportal.
PENTING: JANGAN Login dulu! Biarkan kolom username/password kosong atau terisi, tapi jangan klik tombol Masuk.

Langkah 2: Buka Developer Tools

Tekan tombol F12 di keyboard Anda (atau Klik Kanan > Inspect) untuk membuka panel Developer Tools. Pilih tab Network.

Langkah 3: Lakukan Login

Setelah tab Network terbuka, silakan isi username & password Anda di halaman web, lalu klik tombol Masuk / Login.

Langkah 4: Filter "signin" & Salin Token

Di daftar Network, cari nama request signin (gunakan kotak filter jika perlu). Klik request tersebut.

Lihat di panel kanan (tab Preview atau Response). Cari teks accessToken: "...". Salin kode panjang di dalamnya.

Contoh Token: eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJ1dWlkIjoiNzY3MDM4NmQt...
Ilustrasi: Tab Network > Filter "signin" > Response > Ambil accessToken
⚠ Penting: Token ini bersifat sementara (sekitar 1-2 jam). Jika kadaluarsa, ulangi langkah ini.
